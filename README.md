## How to Open the Site in a Browser

### Option 1: Open Directly (Local File)

1. **Locate the project folder** on your computer:

2. **Double-click `index.html`** - it will open automatically in your default browser

OR

**Right-click `index.html`** → **Open with** → Choose your browser (Chrome, Firefox, Edge, etc.)

### Option 2: Using VS Code Live Server

1. **Open the project in VS Code**
2. **Install Live Server extension** (if not already installed)
3. **Right-click `index.html`** → **"Open with Live Server"**
4. The site will open at `http://127.0.0.1:5500`
